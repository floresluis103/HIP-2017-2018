Extras

The ability to learn new programs isn't as easy as I first initially thought in 10th grade when I used Robot C, a pretty simple program used to program the VEX Robotics Kit.

The learning of different languages just excites me, whether its Object-Oriented or Procedural. That small marble sorter that I built was a great experience for me, since it pretty much set me for one field of engineering, rather then being confused in which field I wanted to be like I was at first.

The first small experience in Robot C was pretty exciting to me that it resulted in me spending my free time trying to learn new languages. Those languages being:

  • Java
  • C
  • Python
  • HTML/CSS
  • Ruby
  • Erlang
