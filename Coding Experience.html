Coding Experience

The toughest part about coding wasn't learning how to write the program, it was perservering through the constant mistakes. Anyone can teach somebody a language, but no one can teach someone to be patient through writing code. The toughest part about coding for me was not giving up every time I read error on the Arduino IDE.

I did have prior knowledge of programming before HIP, but it wasn't as high of a level as it is in my second year of the HIP program. I started off in ninth grade trying to understand Java, and my first true coding project was in the tenth grade using Robot C, trying to build a marble sorter.

I knew coding was a tidious task, and knew it requires dedication and time to finish what is envisioned, and although many people question my choice of future career, I always tell them it's because they do not understand the satisfaction that one feels when their program can finally be downloaded without an error preventing it to do so.
